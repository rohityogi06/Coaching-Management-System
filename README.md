# 🎓 Coaching Management System

A modern **Coaching Management System** developed using **Node.js, Express.js, MySQL, EJS, and MVC Architecture**. This project helps coaching institutes manage students, courses, attendance, fees, results, notes, and notices through dedicated **Admin** and **Student** dashboards.

---

## 🚀 Features

### 👨‍💼 Admin Panel

- 🔐 Secure Admin Login
- 👨‍🎓 Manage Students
- 📚 Manage Courses
- ✅ Approve / Reject Student Registration
- 📅 Attendance Management
- 💰 Fees Management
- 📝 Results Management
- 📢 Notice Management
- 📄 Upload PDF Notes
- 📊 Generate PDF Reports
- 👤 Update Admin Profile
- 🔑 Change Password
- 📧 Email Notifications

---

### 👨‍🎓 Student Panel

- 🔐 Student Login
- 👤 View Profile
- 📚 View Assigned Courses
- 📅 View Attendance
- 📊 Attendance Summary
- 💰 View Fees
- 📝 View Results
- 📢 View Notices
- 📄 Download Notes
- 📞 Contact Coaching

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/rohityogi06/Coaching-Management-System.git
```

Go to project folder

```bash
cd Coaching-Management-System
```

Install dependencies

```bash
npm install
```

Create a `.env` file

```env
EMAIL_USER=your_email@gmail.com
EMAIL_PASS=your_app_password
```

Start the server

```bash
npm start
```
